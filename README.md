# Exam Planner

## Project Overview
Exam Planner is a Java-based desktop application developed for Izmir University of Economics.
The system is designed to schedule exam intervals efficiently by managing students, courses,
classrooms, and attendance lists.

The application works as a standalone system and does not require any external services,
internet connection, or additional software installations other than Java Runtime Environment.

## Features
- Import student, course, classroom, and attendance list data from CSV files
- Edit imported data directly within the application
- Remove students, courses, classrooms, and attendance records
- Re-import updated CSV files by replacing or merging existing data
- Automatic exam scheduling without imposing limitations on the user
- User-friendly JavaFX-based graphical interface
- Built-in Help/About section

## Functional Requirements
- The user can import CSV files containing students, courses, classrooms, and attendance data
- The user can edit all imported data within the application
- The user can remove any existing data from the system
- The user can re-import updated CSV files at any time

## System Requirements
- Java Runtime Environment (JRE) 8 or higher
- No additional software installation is required

## Installation
1. Download the setup file from the **Releases** section of this repository.
2. Double-click the setup file.
3. Follow the installation steps.
4. Run the application.

## Usage
After installation, the application can be launched from the installation directory or
desktop shortcut. Users can import CSV files and manage exam scheduling through the menu
interface.

## Help
The application includes an internal Help/About menu that provides information about the
system and its usage.
